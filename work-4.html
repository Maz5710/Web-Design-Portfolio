TECHNOLOGIES

DISCLAIMER

As a group (four students) worked on a collaborative journey to develop an intuitive application that effectively presents data from an API in a user-friendly manner. The primary goal is to create a solution that allows users to comprehend and interact with the data effortlessly.


Before diving into the implementation phase, meticulous planning takes place, involving modeling the application and conducting prototype testing. These steps ensure a clear understanding of the desired functionality and enable the team to gather valuable user feedback early on.


Throughout the development process, the team followed agile methodologies to manage the project effectively. This approach emphasises iterative development, constant communication, and adaptability. Regular feedback loops and incremental improvements are key aspects of the agile workflow, allowing the team to respond swiftly to changing requirements and user needs.


Key features: Niche market app using MongoDB to store product and user data and bycrypt to secure passwords
