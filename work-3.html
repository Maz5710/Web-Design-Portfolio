Explore Transport Campaign Website

OVERVIEW

TECHNOLOGIES

DISCLAIMER

This website campaign as an extension of Tourism New Zealand’s application using their branding theme and feel.


This project was to create a user-friendly single-page web application for an enhanced visitor experience. The application will allow users to input relevant information, such as the number of people in the party and the duration of users intended travel. Based on user input, the application will dynamically present suitable transport options to me.


In addition, the application will facilitate the calculation of transport costs, taking into account factors such as the chosen vehicle type and the distance to be traveled. This calculation will include an estimation of fuel consumption to provide me with accurate cost estimates.


Through this web application, the user will have a seamless experience, with the ability to input information, receive tailored transport options, and calculate precise cost estimates based on my specific requirements.


Key features: Incorporated Mapbox as an alternative to Google Maps.
