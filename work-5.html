DISCLAIMER

The Star Safari website is an innovative CMS-based platform that caters to both end users and content administrators. With a focus on delivering a seamless user experience, the website features a custom theme incorporating a blend of custom and third-party functionalities. It ensures compatibility across various browsers, devices, and platforms.


To meet the needs of both end users and web content administrators, the website provides specific post and page types. These components empower web administrators to easily manage the website. Additionally, an industry-standard agreement is assembled prior to production, ensuring a clear understanding between the client and the website development team.


To support web administrators, comprehensive guideline documentation was provided, offering guidance on managing and maintaining the website. Moreover, a training session was conducted for the client to familiarise them with the content management system, empowering them to efficiently utilise its features.


Key features: Custom WordPress heme using React
